FFMPEG
~~~~~~

Follow http://www.renevolution.com/how-to-install-ffmpeg-on-mac-os-x/:

- Install XCode
- Install Commandline Tools if necessary
- install home brew `ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"`
- do `brew doctor`
